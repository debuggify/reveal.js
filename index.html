Understanding Web Development

A modern Web development involves

  • Bleeding Edge Browsers like Internet Explorer, Chrome, Firefox, Safari, Opera and many more

The Problem

Due the dynamic nature of the web, NO ONE can guarantee that a website working today will be working tomorrow


If things breaks, it is hard to answer WHEN, WHERE, WHY, WHAT(its impact?) & WHICH (to fix first).

The goal

We want to help developers build rich web apps, in a compelling development environment

What we provide?

We are cloud based debugging service that make life of developers easy
We help to identify to the problem and solve them.

Identifaction of problem

When?

  • Notify developers through email about any error observed on their website
  • Developers can create custom alerts on the type of errors

Where?

  • Provide details like geo location
  • Report the browser version and Operating System
  • Provide overall statistics about the website analytics

Why?

  • Give technical stack related information to analye the problem
  • Provides all instance variables at the time when

What?

  • Type of issue - errors or warnings
  • We tell the developers impact of the errors on their website
  • Statistics like frequncy of the problem, unique user affect per problem helps the developers priortize the problems

Solution

  • We provide developers a simple logging library through which they can add debugging messages
  • Wheneve some issue is logged across all platform these message gets accumulate on a central server and user gets notified
  • Developers can visit their dashboard which provides all information need to debug the problem
  • This service saves lot of time for regression testing hence improving the development experience
  • Service also focuses on team collaboration during development and anything that make the developers more productive
  • The market for web-based technologies is growing very fast, so will be our service
  • Practically any page on web can use this service for getting bug free experience
  • Serivce will be primarliy used by developers, bloggers and big companies

var supports3DTransforms =  document.body.style['webkitPerspective'] !== undefined ||
				document.body.style['MozPerspective'] !== undefined ||
				document.body.style['perspective'] !== undefined;

function linkify( selector ) {
    if( supports3DTransforms ) {

        var nodes = document.querySelectorAll( selector );

        for( var i = 0, len = nodes.length; i < len; i++ ) {
            var node = nodes[i];

            if( !node.className || !node.className.match( /roll/g ) ) {
                node.className += ' roll';
                node.innerHTML = '' + node.innerHTML + '';
            }
        };
    }
}
