Web 101

What is DEBUGGIFY ?


Cloud based debugging service for web developers which help them to debug their websites more easily.

How it works ?

Sites often breaks on different browsers, so debuggify tracks browser related errors

There are many languages, libraries & frameworks, So debuggify tools helps to track errors across different technologies

javascript php coffeescript rails ruby java erlang handlebars backbone python django spring jquery nodejs zend wordpress drupal

Web also depends on third party apis it also helps to debug 3rd party api's

Facebook Google Twitter Github Maps Instagram Pinterest Amazon Delicious Flickr foursquare

If something breaks, Its hard to answer the questions

  • When it breaks ?
  • How many times it breaks ?
  • How many users are facing the issues
  • What is the reason?
  • What is impact of the problem ?

Debuggify helps to answer these question easily
